# OAU Law Reports (OAULR) - Frontend Repository

This is the official frontend repository for the **OAU Law Reports (OAULR)**, the digital legal repository for the Faculty of Law, Obafemi Awolowo University, Ile-Ife.

This project is built with **Next.js 14 (App Router)**, **TypeScript**, and **Tailwind CSS**, featuring a clean "Light Theme" academic aesthetic and smooth animations via **Framer Motion**.
